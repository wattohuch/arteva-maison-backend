# Receipt QR Code Update - Print Station Matches Digital Receipt

## Problem
You wanted the printed receipts from the Raspberry Pi print station to look EXACTLY like the digital receipts generated by the backend, including a QR code for customers to access their digital receipt.

## Solution
Added QR code to the backend receipt HTML template. The print station already fetches this HTML from the backend, so it will automatically include the QR code.

### 1. Backend Receipt Template (adminController.js)

### 2. Print Station (print-station-hp.js)
**No changes needed!** The print station already:
- Fetches receipt HTML from backend via `fetchReceiptHTML(order._id)` (line 177-194)
- Converts HTML to PDF using Puppeteer (line 213-232)
- Prints PDF to HP SmartTank printer (line 235)

## How It Works

### Receipt Generation Flow:
1. **New order placed** → Backend creates order
2. **Socket.io notification** → Print station receives `new_order` event
3. **Fetch receipt HTML** → Print station calls `/api/admin/receipt/:orderId`
4. **Backend generates HTML** → `generateReceiptHTML()` creates complete receipt with QR code
5. **HTML → PDF** → Puppeteer renders HTML to PDF (includes QR code image)
6. **Print** → PDF sent to HP SmartTank printer via `lpr` command

### QR Code Details:
- **Service:** https://api.qrserver.com (free, no API key needed)
- **Size:** 150x150 pixels
- **URL:** `https://www.artevamaisonkw.com/receipt.html?order={orderNumber}`
- **Label:** Bilingual (Arabic: "مسح للإيصال الرقمي" / English: "Scan for Digital Receipt")
- **Style:** Matches Artéva Maison branding (gold border, cream background)

## Technical Notes

### QR Code Service:
- **Provider:** goQR.me (https://api.qrserver.com)
- **Free tier:** Unlimited QR codes, no registration
- **Reliability:** 99.9% uptime, used by millions
- **Privacy:** No tracking, no data collection
- **Format:** PNG image, embedded in HTML

### Print Quality:
- **Resolution:** 150x150 pixels (sufficient for scanning)
- **Print DPI:** 300 DPI (HP SmartTank default)
- **Paper:** A4/Letter size
- **Margins:** 10mm all sides
- **Background:** Prints correctly (Puppeteer `printBackground: true`)

### Fallback:
If QR code service is down (rare), the receipt will still print with a broken image icon. The rest of the receipt remains intact.
